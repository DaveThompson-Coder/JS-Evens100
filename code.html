                            // get the values from the page
                            //Start or Controller function
                            function getValues() {
                                //get values from the page
                                let startValue = document.getElementById("startValue").value;
                                let endValue = document.getElementById("endValue").value;
                            
                                //Inputs MUST be Validated so you know they are integers!!
                                //by using parse into Integers function
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);
                            
                                if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                                    //call the generateNumbers function
                                    let numbers = generateNumbers(startValue, endValue);
                                    //call the displayNumbers function
                                    displayNumbers(numbers);
                                } else {
                                    alert("You must enter integers");
                                }
                            }
                            
                            // generate numbers from the start value to the end value
                            //These are the Logic Function(s)
                            function generateNumbers(sValue, eValue) {
                            
                                let numbers = [];
                            
                                //we want to get all from start to end
                            
                                for(let index = sValue; index <= eValue; index++){
                                    //this will exercute in a loop until index = eValue
                                    numbers.push(index);
                                }
                                return numbers;
                            }
                            
                            //display the numbers and mark the even numbers bold
                            //display or view function
                            function displayNumbers(numbers) {
                            
                                let templateRows = "";
                                for (let index = 0; index < numbers.length; index++) {
                            
                                    let className = "even";
                            
                                    let number = numbers[index];
                            
                                    if(number % 2 == 0) {
                                        className = "even";
                                    }
                                    else {
                                        className = "odd";
                                    }
                                    //This code is a template literal string & doesn't render correctly in 
                                    //Prism, because it has HTML not just JS. Please view source code.  
                                    templateRows += `${number}`;
                                }    
                                document.getElementById("results").innerHTML = templateRows
                            }
Code Structure for Evens100 App.
The getNumber() function

This function gets the start and end values for the app. The values are Parsed to check they are integers. If they are integers, this function calls the generateNumbers(), and the displayNumbers() functions. Otherwise, it displays an alert stating that the input values must be integers.

The generateNumbers() function

This function sets numbers variable to an empty list. Then it runs a for loop from the startValue to the endValue input by the user. The numbers are pushed into the numbers list. When done, it returns the numbers list to the getNumber() function.

The displayNumbers() function

This function runs a for loop from 0 to the length of the numbers list, to show a row in the results table for each value in the numbers list. Checks if the each of the numbers are even or odd. If it is even, it calls the class "even" which changes the font weight to display the number in bold, else it sets the classname to "odd". It adds the number formatted in html to a template literal string called templateRows, it adds this completed string into a table in HTML to be displayed.
